## Todo Task Web Aplication

## About The Project

Todo Task Manager is a full-stack web application developed to help users efficiently organize and manage their daily tasks. The application provides a simple and intuitive interface where users can create, update, complete, and delete tasks according to their requirements.

The primary objective of this project is to simplify task management and improve productivity by allowing users to keep track of their pending and completed activities in one place. The application follows CRUD (Create, Read, Update, Delete) operations and demonstrates the practical implementation of full-stack web development concepts.

The frontend of the application is built using HTML and CSS, providing a clean, responsive, and user-friendly interface. The backend is developed using Node.js and Express.js, which handle server-side logic, API routing, and communication with the database. MongoDB is used as the database to store and manage task-related information efficiently.

## Features
